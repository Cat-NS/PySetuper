# PySetuper
##           Installing
1. Download source code or release
2. Open setup.bat
3. In there enter y / n (Python installed?)
4. Wait... (If you enter n on step 3 just install python in new window)
5. Open cmd and enter %apper% or go to app dir and start appsetuper.py
##       Recommended
1. App dir: C:\Program Files\
2. Use python 3.9+
